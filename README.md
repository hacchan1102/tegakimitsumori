# 手書き見積書 → Excel 変換ツール（公開用）

対応形式：JPG・PNG・**PDF**（1ファイル＝1見積書として処理）

## 精度向上の仕組み
- 手書き数字で間違えやすいパターン（0/6、1/7、3/8など）を明示して読み取らせています。
- 各行の「数量×単価＝金額」を内部で検算させ、原本の税込合計と自動照合します。
- 合計が合わない場合だけ、**同じ画像をもう一度見返して誤読箇所を探す「検算パス」を自動で追加実行**します（この場合のみAPI呼び出しが1件につき2回になります＝料金も2回分）。
- 「品名が怪しい」のか「金額が怪しい」のかを分けて黄色マークで表示します。
- どちらの対策をしても原本自体が不鮮明な場合は、無理に一致させず「要確認」のまま残します。最終的な金額は必ず原本と目視確認してください。

## 構成
```
deploy/
├─ index.html        ← Webページ本体（/api/extract を呼ぶ設定済み）
└─ api/
   └─ extract.js     ← APIキーを隠す中継（Vercelサーバーレス関数）
```

## Vercelで公開する手順

1. このまま `deploy` フォルダを Vercel にデプロイ
   - 一番早いのは **Vercel Drop**（https://vercel.com/new → フォルダをドラッグ）
   - または Vercel CLI：`cd deploy && vercel`
2. Vercel のプロジェクト設定 → **Settings → Environment Variables** に追加
   - 名前: `ANTHROPIC_API_KEY`
   - 値: あなたのAnthropic APIキー（`sk-ant-...`）
3. 再デプロイ（環境変数を反映）

これで、誰でもURLを開いて「画像アップロード → Excel書き出し」ができます。
APIキーはサーバー側（extract.js）だけが使い、ブラウザには出ません。

## 仕組み
- ブラウザが画像を縮小 → `/api/extract` に送信
- `extract.js` がAPIキーを付けてAnthropicへ中継し、読み取り結果を返す
- ブラウザ側で内容を表示・修正 → ExcelJSで整形Excelを生成しダウンロード

## 注意
- API利用にはAnthropicの従量課金が発生します（画像1枚あたり数円程度）。
- 公開URLをそのまま不特定多数に渡すと、その分の利用料が発生します。
  社内限定にする場合は Vercel の Password Protection や、簡易パスコード入力などの追加を検討してください。
- 読み取り結果は必ず原本と照合してください（特に「要確認」表示の行）。

## テスト
このチャットのプレビューで動く版（`index.html`、Anthropic直叩き）でも、
読み取り〜Excel書き出しの一連を確認できます。本番公開時は `deploy/` 側を使ってください。
